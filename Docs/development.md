# 開発について
## 主要ライブラリのバージョン
```
react@16.13.1
react-native@0.63.2
expo@42.0.4
typescript@4.0.8
```
## 環境構築
1. インストール
```
npm install
```

2. expoプロジェクトstart
```
npm start
```

## ルール
### ブランチについて
- `main` 本番環境用のブランチ
- `develop` 開発環境用のブランチ
### issue（チケット）の切り方
- 一つのチケットには一つの修正内容について議論するようにしてください。
### ブランチの切り方
- 一つのブランチにはチケット内に記述されている変更のみ含むようにしてください。
### Pull Requestの出し方
- `develop`ブランチをターゲットに作成してください。
- PRとチケットはできるだけ１対１の関係であるようにしてください。（１対多はできるだけ避けてください）
